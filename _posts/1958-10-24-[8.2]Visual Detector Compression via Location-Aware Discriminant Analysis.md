# [8.2] Visual Detector Compression via Location-Aware Discriminant Analysis

- Authors: Qizhen Lan, Jung Im Choi, Qing Tian
- [arXiv Link](https://arxiv.org/abs/2509.17968v1)
- [PDF Link](https://arxiv.org/pdf/2509.17968v1.pdf)

## Subfields
 感知 / 目标检测与模型压缩 (Perception / 2D Object Detection & Model Compression)
## Reason for Interest

论文提出了一种结合位置感知判别分析（LDT）的剪枝方法，针对自动驾驶中极为关键的边缘端部署效率问题提供了有效解决方案。1. **创新性**：将LDA（线性判别分析）思想引入深度网络剪枝，并结合物体位置信息（Location-aware）来最大化和压缩判别特征，理论具有新意且符合检测任务特性。2. **实验效果**：在KITTI（自动驾驶标准集）和COCO上验证充分，不仅大幅降低了FLOPs和延迟（CPU延迟降低36%），而且在部分设定下出现了‘剪枝后精度反而提升’的现象（如YOLOX-S提升1.6% mAP），证明了方法的正则化优势。3. **行业价值**：虽然主要针对2D检测，但高效的视觉感知是自动驾驶系统的基石，该方法对车端嵌入式部署具有很高的实用价值。扣分点在于仅限于2D检测，未涉及3D检测或BEV架构的验证。
