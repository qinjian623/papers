# [8.6] LaneDiffusion: Improving Centerline Graph Learning via Prior Injected BEV Feature Generation

- Authors: Zijie Wang, Weiming Zhang, Wei Zhang, Xiao Tan, Hongxing Liu, Yaowei Wang, Guanbin Li
- [arXiv Link](https://arxiv.org/abs/2511.06272v1)
- [PDF Link](https://arxiv.org/pdf/2511.06272v1.pdf)

## Subfields
 在线高精地图构建 / 车道线拓扑预测 (Online HD Map Construction / Lane Graph Learning)
## Reason for Interest

1. 创新性（8.5/10）：论文提出了一种独特的生成式范式（LaneDiffusion），与以往直接生成矢量坐标的方法（如 PolyDiffuse）不同，该方法创新性地在 BEV 特征层面上应用扩散模型，利用先验注入模块（LPIM）和扩散模块（LPDM）来优化特征表示，从而有效解决了复杂场景下的遮挡和模糊问题。这种'特征级扩散'的思路在地图学习领域较为新颖。
2. 实验完整性与 SOTA（9.0/10）：在 nuScenes 和 Argoverse2 两个权威数据集上进行了广泛的测试，且在点级（Point-level）和段级（Segment-level）指标上均全面超越了 MapTR、VectorMapNet 和 CGNet 等强基线模型，提升幅度显著（如 APLS 提升 6.4%），SOTA 声明有坚实的数据支撑。
3. 可信度与行业潜力（8.0/10）：论文逻辑严密，消融实验详细验证了各个模块的有效性。主要顾虑在于扩散模型的推理延迟（虽然论文使用了 T=15 的采样步数并讨论了效率，但相比单步回归模型仍有计算代价），这在车端实时部署中可能是一个瓶颈。但总体而言，该方法为解决长尾感知问题提供了高质量的解决方案。
