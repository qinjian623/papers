# [9.2] Register Any Point: Scaling 3D Point Cloud Registration by Flow Matching

- Authors: Yue Pan, Tao Sun, Liyuan Zhu, Lucas Nunes, Iro Armeni, Jens Behley, Cyrill Stachniss
- [arXiv Link](https://arxiv.org/abs/2512.01850v1)
- [PDF Link](https://arxiv.org/pdf/2512.01850v1.pdf)

## Subfields
 3D感知 / SLAM (点云配准与建图)
## Reason for Interest

1. 强创新性：摒弃了传统的‘两两配准+姿态图优化（PGO）’范式，提出基于Flow Matching的单阶段生成式配准框架，利用刚性约束（Rigidity-forcing）解决生成模型的漂移问题。
2. 泛化能力极强：构建了包含17个数据集（如KITTI, NuScenes, Waymo等）的超大规模训练集（100k+样本），实现了跨传感器（LiDAR/深度相机）和跨场景（室内/室外）的零样本（Zero-shot）泛化。
3. 解决痛点：在自动驾驶极具挑战的低重叠率（Low-overlap）和多视角建图场景下，性能大幅超越现有几何和深度学习方法，对高精地图构建、多车协同SLAM及回环检测具有极高的行业应用价值。
4. 局限性：推理速度（秒级）目前更适合后端优化或离线建图，尚未达到前端里程计的实时性要求，但不影响其作为通用的后端配准底座的价值。
