# [8.9] Monocular 3D Lane Detection via Structure Uncertainty-Aware Network with Curve-Point Queries

- Authors: Ruixin Liu, Zejian Yuan
- [arXiv Link](https://arxiv.org/abs/2511.13055v1)
- [PDF Link](https://arxiv.org/pdf/2511.13055v1.pdf)

## Subfields
 3D Lane Detection / Perception
## Reason for Interest

论文提出 MonoUnc 框架，针对单目 3D 车道线检测中的偶然不确定性（Aleatoric Uncertainty）问题，创新性地将局部车道线段建模为 3D 高斯分布，并设计了 3D 高斯匹配损失，显著提升了对噪声的鲁棒性。此外，论文指出了现有评价指标（单向倒角距离）的缺陷（如对预测过长不敏感），提出了更严格的双向倒角距离（Bidirectional Chamfer Distance）指标，不仅在方法论上有贡献，在评价体系上也有改进。实验在两大主流数据集上均取得 SOTA，且代码开源，具有较高的行业应用潜力和研究价值。
