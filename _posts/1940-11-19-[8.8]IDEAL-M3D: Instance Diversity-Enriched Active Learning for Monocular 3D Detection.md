# [8.8] IDEAL-M3D: Instance Diversity-Enriched Active Learning for Monocular 3D Detection

- Authors: Johannes Meier, Florian Günther, Riccardo Marin, Oussema Dhaouadi, Jacques Kaiser, Daniel...
- [arXiv Link](https://arxiv.org/abs/2511.19301v1)
- [PDF Link](https://arxiv.org/pdf/2511.19301v1.pdf)

## Subfields
 Monocular 3D Object Detection / Active Learning
## Reason for Interest

1. **切中痛点与理论洞察**：论文敏锐地指出了传统基于不确定性（Uncertainty-based）的主动学习在单目 3D 检测中的致命缺陷——倾向于选择深度歧义大但在语义上信息量低的远距离目标。提出基于实例级（Instance-based）的多样性选择策略，有效解决了这一偏差。
2. **创新性**：提出了 IDEAL-M3D 框架，结合了异构集成（Heterogeneous Ensembles）与任务无关的视觉特征（利用 SAMv2 和 Stable Diffusion 提取），在特征空间覆盖上通过几何与外观的双重互补性实现了高效样本挖掘。同时提出了 NAURC 指标，解决了基于图像和基于实例的主动学习方法难以公平对比的问题。
3. **实验充分且结果显著**：在 KITTI、Waymo、Rope3D 三个主流数据集上验证了方法的有效性。特别是“用 60% 数据达到全监督效果”的结论对于自动驾驶行业降低高昂的 3D 标注成本具有极高的实用价值。
4. **潜力与局限**：虽然训练阶段引入集成模型增加了计算开销（尽管论文提出了轻量化改进），但考虑到标注成本的节省，这是一个合理的 Trade-off。该方法对数据闭环（Data Loop）建设具有直接指导意义。
## Abstract: 
  To address these limitations, we propose IDEAL-M3D, the first instance-level pipeline for monocular 3D detection. For the first time, we demonstrate that an explicitly diverse, fast-to-train ensemble improves diversity-driven active learning for monocular 3D. We induce diversity with heterogeneous backbones and task-agnostic features, loss weight perturbation, and time-dependent bagging. IDEAL-M3D shows superior performance and significant resource savings: with just 60% of the annotations, we achieve similar or better AP3D on KITTI validation and test set results compared to training the same detector on the whole dataset.
