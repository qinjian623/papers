# [8.0] SAMIRO: Spatial Attention Mutual Information Regularization with a Pre-trained Model as Oracle for Lane Detection

- Authors: Hyunjong Lee, Jangho Lee, Jaekoo Lee
- [arXiv Link](https://arxiv.org/abs/2511.10385v1)
- [PDF Link](https://arxiv.org/pdf/2511.10385v1.pdf)

## Subfields
 感知 / 车道线检测 (Lane Detection)
## Reason for Interest

论文提出了一种即插即用（Plug-and-Play）的正则化方法 SAMIRO，利用大规模预训练模型（MIM）作为 Oracle 进行知识蒸馏，引入空间注意力机制以保留车道线结构特征。核心价值在于：1）零推理成本提升性能，非常契合车端部署需求；2）在阴影、夜间等困难场景下表现出较好的鲁棒性；3）在三个主流数据集上均有一致性提升。虽然整体指标提升幅度较小（<1%），但实验设计严谨，工程实用价值较高。
