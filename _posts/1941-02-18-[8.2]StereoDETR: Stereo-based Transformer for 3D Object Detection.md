# [8.2] StereoDETR: Stereo-based Transformer for 3D Object Detection

- Authors: Shiyi Mu, Zichong Gu, Zhiqi Ai, Anqi Liu, Yilin Gao, Shugong Xu
- [arXiv Link](https://arxiv.org/abs/2511.18788v1)
- [PDF Link](https://arxiv.org/pdf/2511.18788v1.pdf)

## Subfields
 3D感知 / 双目立体视觉3D目标检测
## Reason for Interest

1. 创新性与实用性：论文针对双目3D检测计算成本高、延迟大的核心痛点，提出了一种基于Transformer的高效架构。通过解耦单目2D检测分支和双目深度分支，并采用稀疏深度采样策略，成功实现了实时推理（>50 FPS），这在双目领域具有显著的工程价值。
2. 实验表现：在KITTI数据集上，该方法在行人和骑行者等非刚性、小目标类别上表现优异，超越了现有的SOTA方法（如DSC3D）。虽然在Car类别的“中等”难度上略低于最强非实时方法，但在“困难”模式下表现出极强的鲁棒性，且速度快4-6倍，极大提升了端侧部署潜力。
3. 完整性与可信度：代码开源，实验对比详尽（包括速度与精度的Trade-off分析）。
4. 局限性：仅在KITTI数据集上进行了验证，缺乏在nuScenes或Waymo等更大规模、更现代数据集上的实验结果，略微降低了其在当前自动驾驶研究前沿的普适性验证权重。
