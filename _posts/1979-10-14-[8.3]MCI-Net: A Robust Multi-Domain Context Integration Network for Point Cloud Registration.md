# [8.3] MCI-Net: A Robust Multi-Domain Context Integration Network for Point Cloud Registration

- Authors: Shuyuan Lin, Wenwu Peng, Junjie Huang, Qiang Qi, Miaohui Wang, Jian Weng
- [arXiv Link](https://arxiv.org/abs/2512.23472v1)
- [PDF Link](https://arxiv.org/pdf/2512.23472v1.pdf)

## Subfields
 3D Perception / Point Cloud Registration
## Reason for Interest

论文提出了 MCI-Net，一种结合全局图聚合（GNAM）和多域上下文交互（PCIM）的点云配准网络，有效解决了低重叠率下的特征匹配问题。实验验证非常充分，涵盖室内 RGB-D 和室外 LiDAR 场景（KITTI），证明了其在自动驾驶定位与建图中的应用价值。虽然在 KITTI 数据集上大幅降低了平移误差（RTE），但在困难场景（3DLoMatch）下相较于最新的 2025 年同期工作（如 PSReg）提升幅度较小（<0.5%）。此外，单帧耗时 0.23s（RTX3090），对于车端实时里程计（通常需 >10Hz）略显吃力，更适合用于回环检测或离线建图。
