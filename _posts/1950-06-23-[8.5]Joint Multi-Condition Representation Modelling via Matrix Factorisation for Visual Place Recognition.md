# [8.5] Joint Multi-Condition Representation Modelling via Matrix Factorisation for Visual Place Recognition

- Authors: Timur Ismagilov, Shakaiba Majeed, Michael Milford, Tan Viet Tuyen Nguyen, Sarvapali D. Ra...
- [arXiv Link](https://arxiv.org/abs/2510.17739v1)
- [PDF Link](https://arxiv.org/pdf/2510.17739v1.pdf)

## Subfields
 视觉定位 (Visual Localization) / VPR / SLAM
## Reason for Interest

1. **创新性与落地价值高**：论文提出了一种无需训练（Training-free）的矩阵分解（QR/SVD）方法来处理视觉定位中的多条件（季节、光照、视角）问题。相比于需要昂贵训练的端到端大模型，这种基于子空间投影的方法计算轻量且易于集成到现有SLAM系统中，非常适合自动驾驶长周期地图维护。
2. **实验验证充分**：在多个经典自动驾驶定位数据集（Oxford RobotCar, Nordland）及非结构化数据集（GLDv2）上进行了广泛测试，并构建了包含多视角和时间跨度的SotonMV新基准，全面验证了方法在视角和外观变化下的鲁棒性。
3. **性能优越**：在不改变底层特征提取器（如CosPlace, MixVPR）的前提下，仅通过数学上的子空间建模就显著提升了检索召回率，证明了方法的有效性和通用性。
4. **行业潜力**：该方法支持地图压缩（SVD降维）和增量更新，对车端有限算力下的高精地图定位具有很高的工程参考价值。
