# [7.8] Calibrating the Full Predictive Class Distribution of 3D Object Detectors for Autonomous Driving

- Authors: Cornelius Schröder, Marius-Raphael Schlüter, Markus Lienkamp
- [arXiv Link](https://arxiv.org/abs/2510.01829v1)
- [PDF Link](https://arxiv.org/pdf/2510.01829v1.pdf)

## Subfields
 3D Perception / Uncertainty Calibration (3D感知 / 不确定性校准)
## Reason for Interest

1. 选题切中安全痛点：论文针对自动驾驶中至关重要的置信度校准问题，特别是指出了现有方法仅校准主导类别（Dominant Class）的不足，提出了全预测分布校准（Full Predictive Class Distribution）的概念，对下游概率规划具有重要意义。
2. 方法创新且实用：提出了新的评估指标（Full D-ECE）和训练时的辅助损失函数（Auxiliary Loss），能够有效提升模型对非主导类别的置信度估计准确性，且实现方案具备通用性，可应用于多种检测器。
3. 实验扎实：在Waymo数据集上对比了三种主流检测器（CenterPoint, PillarNet, DSVT-Pillar）和多种校准方法（温度缩放、Platt缩放等），分析详尽。
4. 局限性：虽然动机是为了提升规划安全性，但实验仅停留在感知层面的校准指标上，缺乏闭环实验来证明该校准改进对最终规划控制策略的具体增益，因此在行业潜力的证明上略有欠缺。
