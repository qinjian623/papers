# [3.8] Goal-Oriented Multi-Agent Reinforcement Learning for Decentralized Agent Teams

- Authors: Hung Du, Hy Nguyen, Srikanth Thudumu, Rajesh Vasa, Kon Mouzakis
- [arXiv Link](https://arxiv.org/abs/2511.11992v1)
- [PDF Link](https://arxiv.org/pdf/2511.11992v1.pdf)

## Subfields
 多智能体强化学习 (MARL) / 协同规划
## Reason for Interest

论文提出了一种基于目标感知的去中心化MARL权重共享机制，旨在解决网联车在有限通信下的协同问题。然而，实验完全基于简单的2D网格世界（Grid World），仅涉及上下左右移动，缺乏车辆动力学、真实交通流、传感器输入等自动驾驶核心要素。虽然理论上针对CAV（网联自动驾驶），但其实验环境过于抽象，距离实际车端应用或具有行业参考价值的仿真（如CARLA/SUMO）差距巨大，属于理论探索阶段，因此依据评分规则严格限制在5分以下。
