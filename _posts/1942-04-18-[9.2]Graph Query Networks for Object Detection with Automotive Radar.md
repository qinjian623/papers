# [9.2] Graph Query Networks for Object Detection with Automotive Radar

- Authors: Loveneet Saini, Hasan Tercan, Tobias Meisen
- [arXiv Link](https://arxiv.org/abs/2511.15271v1)
- [PDF Link](https://arxiv.org/pdf/2511.15271v1.pdf)

## Subfields
 3D感知 / 4D毫米波雷达目标检测
## Reason for Interest

1. 创新性高：针对雷达点云稀疏、不规则的痛点，提出了Graph Query Networks (GQN)，利用动态图查询（Graph Queries）在BEV空间中自适应采样并构建对象级图结构，这种从'网格/序列处理'转向'图查询推理'的思路非常契合雷达特性。
2. 性能卓越：实验结果极具说服力。作为一个即插即用的模块，GQN能显著提升不同基线的性能（例如将PointPillars的mAP从22.0%提升至33.7%，相对提升+53.1%），最终达成的42.4% mAP在雷达纯感知领域属于极高水平。
3. 实验完整：不仅在nuScenes上进行了详尽的对比和消融实验（如K值敏感性、采样密度），还在私有数据集上分析了对弱势道路使用者（VRU）的检测增益。
4. 行业价值：毫米波雷达是量产自动驾驶的核心传感器，提升其纯感知性能可降低对昂贵激光雷达的依赖，该方法计算成本适中（增加少量FLOPs），具备极高的工程落地潜力。
