# [8.8] SemOD: Semantic Enabled Object Detection Network under Various Weather Conditions

- Authors: Aiyinsi Zuo, Zhaoliang Zheng
- [arXiv Link](https://arxiv.org/abs/2511.22142v1)
- [PDF Link](https://arxiv.org/pdf/2511.22142v1.pdf)

## Subfields
 感知 / 恶劣天气目标检测 (Perception / Adverse Weather Object Detection)
## Reason for Interest

1. 创新性：论文提出了一种语义感知的目标检测框架（SemOD），创新地利用语义分割先验信息同时指导图像去噪（PPU）和目标检测（DTU）两个阶段。特别是通过Attention Embedded Decoder (AED) 和 Domain Adaptation Block (DAB) 有效缓解了恶劣天气下的域偏移问题。
2. 实验完整性：构建了涵盖雾、雨、雪多种天气的综合数据集，对比了包括图像增强+检测、域自适应检测在内的多种主流方案。消融实验清晰展示了语义模块和域适应模块的贡献。
3. 结果可信度：在多个恶劣天气基准上取得了显著的性能提升（尤其是雪天场景），且提供了开源代码链接，结果可复现性高。
4. 行业价值：直接针对自动驾驶感知的长尾痛点（恶劣天气），虽然推理延迟较纯检测器有所增加（从~10ms增至~40ms），但仍在实时范围内（~25FPS），具有较高的工程落地潜力。
