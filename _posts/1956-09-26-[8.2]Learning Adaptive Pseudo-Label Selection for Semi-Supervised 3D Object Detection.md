# [8.2] Learning Adaptive Pseudo-Label Selection for Semi-Supervised 3D Object Detection

- Authors: Taehun Kong, Tae-Kyun Kim
- [arXiv Link](https://arxiv.org/abs/2509.23880v1)
- [PDF Link](https://arxiv.org/pdf/2509.23880v1.pdf)

## Subfields
 3D感知 / 激光雷达目标检测 (Semi-Supervised Learning)
## Reason for Interest

该论文提出了一种可学习的伪标签选择模块（PSM），通过预测伪标签的GT-IoU质量并自适应生成阈值，有效解决了半监督学习中依赖人工固定阈值的问题。创新性方面，将阈值选择从启发式规则转化为学习问题（PQE+CTE模块），逻辑清晰且符合直觉。实验方面，在KITTI极少样本（1%）场景下提升显著，证明了方法在数据稀缺场景下的鲁棒性，具有较高的行业应用价值（降低标注成本）。虽然在Waymo大模型上未完全超越PTPM（归因于PTPM使用了更强的Teacher结构，两者正交），但相比直接基线HSSDA有稳定提升，整体工作扎实可信。
