# [9.2] V2X-RECT: An Efficient V2X Trajectory Prediction Framework via Redundant Interaction Filtering and Tracking Error Correction

- Authors: Xiangyan Kong, Xuecheng Wu, Xiongwei Zhao, Xiaodong Li, Yunyun Shi, Gang Wang, Dingkang Y...
- [arXiv Link](https://arxiv.org/abs/2511.17941v1)
- [PDF Link](https://arxiv.org/pdf/2511.17941v1.pdf)

## Subfields
 V2X 协同轨迹预测 (V2X Cooperative Trajectory Prediction)
## Reason for Interest

该论文具有极高的行业落地价值与学术创新性。1. **切中痛点**：针对 V2X 系统中常见的 ID 切换（ID Switch）问题，提出了多视角一致性匹配与修正模块，直接提升了数据质量，这是以往单纯堆叠模型深度所忽略的工程难题。2. **效率优化**：提出的特征复用（Feature Reuse）机制显著降低了重复编码开销，将推理时间压缩至 100ms 级别，使其具备了实车部署的可行性。3. **机制创新**：利用交通信号灯的变化趋势（不仅仅是状态）来引导交互注意力的筛选，符合交通动力学逻辑。4. **效果显著**：在两个主流数据集上均取得了大幅度的指标提升，且实验对比了 2025 年最新的基线方法（如 Co-MTP, I2XTraj），证明了其优越性。
