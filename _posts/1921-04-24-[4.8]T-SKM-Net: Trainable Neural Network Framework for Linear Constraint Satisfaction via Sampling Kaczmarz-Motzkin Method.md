# [4.8] T-SKM-Net: Trainable Neural Network Framework for Linear Constraint Satisfaction via Sampling Kaczmarz-Motzkin Method

- Authors: Haoyu Zhu, Yao Zhang, Jiashen Ren, Qingchun Hou
- [arXiv Link](https://arxiv.org/abs/2512.10461v1)
- [PDF Link](https://arxiv.org/pdf/2512.10461v1.pdf)

## Subfields
 规划控制 / 基础优化算法 (Planning & Control / Optimization Algorithms)
## Reason for Interest

1. 创新性：提出将采样Kaczmarz-Motzkin (SKM) 方法引入神经网络约束满足层，结合零空间变换处理等式和不等式混合约束，解决了传统方法难以兼顾计算效率与严格约束满足的问题，理论创新性较强。
2. 实验与SOTA：实验在电力系统（DCOPF）和金融投资组合分配任务上表现优异，证明了其在实时性和可行性上的优势，达到了该类通用优化求解方法的SOTA水平。
3. 自动驾驶相关性：尽管摘要和引言提及“自动驾驶”和“机器人路径规划”作为潜在应用场景，但全篇论文没有任何自动驾驶相关的实验（如轨迹规划、避障控制等），也没有使用相关数据集（如nuScenes或仿真环境）。
4. 评分理由：该论文本质是一篇通用的机器学习/运筹优化论文。虽然其提出的快速求解器对自动驾驶中的MPC或轨迹优化有工具价值，但缺乏垂直领域的验证。严格遵循“若不直接和车端自动驾驶相关，最多5分”的规则，给予4.8分。
