# [8.3] Unified Low-Light Traffic Image Enhancement via Multi-Stage Illumination Recovery and Adaptive Noise Suppression

- Authors: Siddiqua Namrah
- [arXiv Link](https://arxiv.org/abs/2511.17612v1)
- [PDF Link](https://arxiv.org/pdf/2511.17612v1.pdf)

## Subfields
 视觉感知 / 低光照图像增强 (Visual Perception / Low-Light Image Enhancement)
## Reason for Interest

论文针对自动驾驶夜间感知的关键痛点（低照度、车灯眩光、运动模糊）提出了一种无监督的多阶段增强框架。其核心创新在于结合了Retinex分解与专门的过曝校正（OEC）模块，有效解决了夜间行车中常见的强光源（如对向远光灯）导致的信息丢失问题，这对于下游的感知任务至关重要。实验验证非常充分，在三个交通专用数据集上均取得了极佳的指标，甚至在部分指标上超越了监督学习方法，显示出极强的数据利用效率和泛化能力。此外，论文提供了42ms的推理速度分析，证明了其在车端部署的实时性潜力。
