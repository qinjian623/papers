# [8.5] BADAS: Context Aware Collision Prediction Using Real-World Dashcam Data

- Authors: Roni Goldshmidt, Hamish Scott, Lorenzo Niccolini, Shizhan Zhu, Daniel Moura, Orly Zvitia
- [arXiv Link](https://arxiv.org/abs/2510.14876v1)
- [PDF Link](https://arxiv.org/pdf/2510.14876v1.pdf)

## Subfields
 自动驾驶安全 / 碰撞预测 (Safety / Collision Prediction)
## Reason for Interest

1. 创新性与价值：论文敏锐地指出了当前碰撞预测研究的痛点——即混淆了‘一般交通异常’与‘自车相关碰撞（Ego-Centric）’。大多数现有数据集包含大量与自车无关的事故（如邻车道碰撞），导致模型在实际部署中误报率极高。论文提出的‘以自车为中心’的问题重构具有极高的行业落地价值。
2. 数据贡献：对主流数据集（DAD, DoTA 等）进行了系统性的重标注，剔除非自车相关事件，并引入了基于人类共识的报警时间（mTTA），这比单纯的二分类指标更符合 ADAS 实际需求。
3. 方法论：利用先进的视频基础模型（V-JEPA2）进行微调，证明了通用的时空表征在特定驾驶安全任务上的有效性。
4. 实验与不足：虽然结果呈现碾压态势（SOTA），但部分原因是测试标准的改变（只测自车事故）天然不利于在原始数据上训练的基线模型。此外，最强模型 BADAS1.0 依赖 40k 私有数据，复现性受限，但开源了基于 1.5k 数据的 BADAS-Open 版本及重标注标签，对社区有积极贡献。
