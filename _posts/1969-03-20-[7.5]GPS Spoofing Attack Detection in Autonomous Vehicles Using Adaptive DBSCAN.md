# [7.5] GPS Spoofing Attack Detection in Autonomous Vehicles Using Adaptive DBSCAN

- Authors: Ahmad Mohammadi, Reza Ahmari, Vahid Hemmati, Frederick Owusu-Ambrose, Mahmoud Nabil Mahmo...
- [arXiv Link](https://arxiv.org/abs/2510.10766v1)
- [PDF Link](https://arxiv.org/pdf/2510.10766v1.pdf)

## Subfields
 自动驾驶安全 / GNSS欺骗检测 (AV Security / GNSS Spoofing Detection)
## Reason for Interest

论文针对自动驾驶中隐蔽性较强的‘渐进式GPS欺骗攻击’提出了有效的解决方案。创新点在于将自适应阈值的 DBSCAN 聚类应用于实时残差分析，具有较强的工程落地潜力（低算力需求，利用现有传感器）。

主要扣分点：
1. 实验验证基于真实数据集上的‘软件模拟攻击’，缺乏物理层面的射频欺骗实测，说服力不如硬件在环测试。
2. 缺失与基准算法（Baseline）的对比分析，难以客观评估其相对于现有技术的具体性能增益。
