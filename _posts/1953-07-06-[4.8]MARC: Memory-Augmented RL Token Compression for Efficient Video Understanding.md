# [4.8] MARC: Memory-Augmented RL Token Compression for Efficient Video Understanding

- Authors: Peiran Wu, Zhuorui Yu, Yunze Liu, Chi-Hao Wu, Enmin Zhou, Junxiao Shen
- [arXiv Link](https://arxiv.org/abs/2510.07915v1)
- [PDF Link](https://arxiv.org/pdf/2510.07915v1.pdf)

## Subfields
 自动驾驶大模型 / 视频理解效率 (Efficient VLM)
## Reason for Interest

论文提出了结合视觉记忆检索与强化学习蒸馏（C-GRPO）的视频Token压缩方法，实现了极高的推理效率（显存降低72%），这对VLM上车部署具有重要价值。然而，论文实验完全局限于通用视频QA基准，缺乏在nuScenes、Waymo或DriveLM等自动驾驶专用数据集上的验证，未直接证明其在车端感知或规划任务中的有效性。根据‘非直接车端相关’的严格评分规则，给予最高档非直接类评分。
