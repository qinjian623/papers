# [4.6] DINO-YOLO: Self-Supervised Pre-training for Data-Efficient Object Detection in Civil Engineering Applications

- Authors: Malaisree P, Youwai S, Kitkobsin T, Janrungautai S, Amorndechaphon D, Rojanavasu P
- [arXiv Link](https://arxiv.org/abs/2510.25140v2)
- [PDF Link](https://arxiv.org/pdf/2510.25140v2.pdf)

## Subfields
 感知 / 2D 目标检测 (长尾/低数据场景)
## Reason for Interest

论文核心聚焦于土木工程场景（裂缝、安全帽检测），虽然使用了KITTI数据集并提及自动驾驶工程车，但其设定的实验基线（YOLOv12仅41% mAP）远低于自动驾驶行业标准（通常>90%），暗示其未利用通用的COCO预训练，这在自动驾驶实践中不符合常规。虽然方法对数据稀缺的特种车辆场景有一定价值，但引入的推理延迟（2-4倍）和绝对精度的不足限制了其在主流车端自动驾驶中的应用潜力。
