# [7.8] Semantic-Drive: Democratizing Long-Tail Data Curation via Open-Vocabulary Grounding and Neuro-Symbolic VLM Consensus

- Authors: Antonio Guillen-Perez
- [arXiv Link](https://arxiv.org/abs/2512.12012v1)
- [PDF Link](https://arxiv.org/pdf/2512.12012v1.pdf)

## Subfields
 自动驾驶数据挖掘 / Data Curation (DataOps)
## Reason for Interest

1. **创新性与价值**：论文针对自动驾驶数据闭环中的核心痛点——“长尾数据挖掘”提出了一套本地化、低成本的解决方案。通过结合符号化感知（YOLOE）与认知推理（VLM）的神经符号架构（Neuro-Symbolic），有效缓解了纯 VLM 的幻觉问题和对小目标的忽视，具有较强的工程实用价值和行业落地潜力（强调在消费级显卡 RTX 3090 上运行）。
2. **方法论**：引入“System 2”慢思考机制和多模型裁判（Judge-Scout）共识策略，逻辑清晰，能够提升数据筛选的准确性和安全性。
3. **实验不足**：定量评估仅基于 116 帧的人工标注（Gold Set）进行，样本量过小，统计显著性较弱。对比基线（Baseline）仅选取了传统的元数据搜索和基础 CLIP 模型，缺乏与同类先进 VLM 检索方法（如 VLMine 等）的直接定量对比，SOTA 含金量有限。
4. **综合评价**：虽然并非端侧实时算法的理论突破，但作为面向 L4 验证的 DataOps 工具，其提出的框架对提升数据迭代效率意义重大。
