# [7.8] Motion-Aware Transformer for Multi-Object Tracking

- Authors: Xu Yang, Gady Agam
- [arXiv Link](https://arxiv.org/abs/2509.21715v2)
- [PDF Link](https://arxiv.org/pdf/2509.21715v2.pdf)

## Subfields
 2D 多目标跟踪 (MOT) / 视觉感知
## Reason for Interest

1. 创新性：针对 DETR 类跟踪器中‘查询冲突’（Query Collision）问题，提出了 Motion-Aware Transformer (MAT) 模块，显式预测 Query 的运动轨迹，而非仅依赖注意力机制的隐式更新，设计思路清晰且有效。
2. 实验完整性：在三个主流数据集上进行了验证，特别是包含自动驾驶场景的 BDD100k，证明了方法的通用性。
3. 可信度：对比了 MOTRv3、MeMOTR 等强基线，消融实验充分证明了 MAT 模块的贡献。
4. 行业价值：虽然主要是 2D 跟踪，但其解决 Transformer 跟踪中 Query 漂移和 ID 频繁切换的思路，对自动驾驶中的纯视觉 3D 跟踪（如 MOTR3D）架构设计具有重要的参考价值。不过限于 2D 空间，对目前主流的 BEV/3D 感知栈的直接集成度略低，因此给予 7.8 分。
