# [8.2] The Overlooked Value of Test-time Reference Sets in Visual Place Recognition

- Authors: Mubariz Zaffar, Liangliang Nan, Sebastian Scherer, Julian F. P. Kooij
- [arXiv Link](https://arxiv.org/abs/2510.03751v1)
- [PDF Link](https://arxiv.org/pdf/2510.03751v1.pdf)

## Subfields
 自动驾驶定位 / 视觉位置识别 (VPR)
## Reason for Interest

该论文切入点非常精准且实用。针对自动驾驶中基于地图的定位场景，利用“地图已知”这一先验条件，提出利用地图数据（Reference Set）进行自监督微调（RSF）。这种方法巧妙地利用了领域内可获得的无标注数据，解决了通用大模型（如DinoV2）在特定环境（如季节变化、不同城市风格）下的域适应问题。实验设计扎实，在多个具有挑战性的数据集上验证了有效性，且证明了微调后的模型仍保持了一定的泛化能力。虽然方法在技术层面主要是流程上的创新而非架构上的突破，但对工程落地（如Robotaxi的城市泛化）具有很高的参考价值和实际提升潜力。
