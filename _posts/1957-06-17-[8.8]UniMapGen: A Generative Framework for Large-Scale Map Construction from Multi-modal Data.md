# [8.8] UniMapGen: A Generative Framework for Large-Scale Map Construction from Multi-modal Data

- Authors: Yujian Yuan, Changjie Wu, Xinyuan Chang, Sijin Wang, Hang Zhang, Shiyi Liang, Shuang Zeng...
- [arXiv Link](https://arxiv.org/abs/2509.22262v2)
- [PDF Link](https://arxiv.org/pdf/2509.22262v2.pdf)

## Subfields
 HD地图构建 / 地图生成 (Map Construction)
## Reason for Interest

1. 创新性：提出基于MLLM（多模态大模型）的生成式地图构建框架，将车端视角（PV）、卫星图（BEV）和文本提示（Text Prompt）结合，有效解决了单纯依赖卫星图存在的遮挡和时效性问题。其提出的'State Update'策略巧妙解决了大范围地图拼接时的几何不连续问题。
2. 实用性与行业价值：该方案大幅降低了高精地图生产对专业采集车的依赖，利用众包数据（PV）更新卫星地图的思路具有极高的落地潜力。支持文本交互式生成也为地图标注员提供了高效工具。
3. 实验完整性：在OpenSatMap等数据集上进行了详尽的对比实验和消融研究，证明了多模态融合对解决遮挡（如树木遮挡道路）的有效性，结果可信度高。
