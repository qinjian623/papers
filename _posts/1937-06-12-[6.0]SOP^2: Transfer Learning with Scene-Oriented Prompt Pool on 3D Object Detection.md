# [6.0] SOP^2: Transfer Learning with Scene-Oriented Prompt Pool on 3D Object Detection

- Authors: Ching-Hung Cheng, Hsiu-Fu Wu, Bing-Chen Wu, Khanh-Phong Bui, Van-Tin Luu, Ching-Chun Huan...
- [arXiv Link](https://arxiv.org/abs/2512.08223v1)
- [PDF Link](https://arxiv.org/pdf/2512.08223v1.pdf)

## Subfields
 LiDAR 3D Object Detection / Transfer Learning (PEFT)
## Reason for Interest

1. **创新性 (Innovation)**: 将 NLP 领域的 Prompt Pool 概念引入 3D 点云检测，提出 Scene-Oriented Prompt Pool (SOP^2) 以适应不同场景，具有一定的理论创新价值。
2. **效果 (Performance)**: 实验结果呈现两面性。在 Pedestrian 类别上，SOP^2 (61.12%) 显著超越了 Full Fine-tuning (55.81%)，证明了其在特定类别迁移上的有效性；但在最关键的 Car 类别上 (82.07%) 仍低于 Full Fine-tuning (83.32%)，且仅比 From Scratch (81.88%) 提升微弱 (+0.19%)。
3. **工程价值 (Industry Value)**: 虽然显著减少了可训练参数 (0.82M vs 8.65M)，但该方法引入了额外的推理延迟 (88ms -> 96ms) 和显存占用。在自动驾驶车端部署中，推理速度和精度通常优于存储空间的节省，因此该方法的实际应用潜力受到延迟增加的限制。
