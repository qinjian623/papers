# [4.8] SPOT!: Map-Guided LLM Agent for Unsupervised Multi-CCTV Dynamic Object Tracking

- Authors: Yujin Noh, Inho Jake Park, Chigon Hwang
- [arXiv Link](https://arxiv.org/abs/2512.20975v1)
- [PDF Link](https://arxiv.org/pdf/2512.20975v1.pdf)

## Subfields
 智能交通系统 (ITS) / 多摄像头跟踪 (Multi-Camera Tracking) / 仿真评估
## Reason for Interest

1. 方向偏差（主要扣分项）：该研究属于路侧感知（Infrastructure-side）和视频监控领域，旨在通过CCTV追踪车辆，而非车端（Vehicle-side）自动驾驶感知或规划算法。根据评分规则，非车端技术最高不超过5分。
2. 创新性：提出了一种基于地图引导的LLM智能体（SPOT!），利用RAG技术结合路网结构进行盲区轨迹预测，思路新颖，具有一定的神经符号学（Neuro-symbolic）参考价值。
3. 实验局限：实验完全基于CARLA仿真，且仅对比了不同LLM和简单启发式规则，缺乏与主流SOTA跟踪器（如DeepSORT变体、FairMOT等）的硬核对比，实际场景下的泛化能力（如应对光照、遮挡、ReID难点）未得到充分验证。
4. 结论：对V2X路侧感知有一定启发，但对车端自动驾驶落地的直接贡献较小。
