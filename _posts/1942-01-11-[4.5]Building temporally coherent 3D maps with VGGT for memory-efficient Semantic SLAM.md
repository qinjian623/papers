# [4.5] Building temporally coherent 3D maps with VGGT for memory-efficient Semantic SLAM

- Authors: Gergely Dinya, Péter Halász, András Lőrincz, Kristóf Karacs, Anna Gelencsér-Horváth
- [arXiv Link](https://arxiv.org/abs/2511.16282v2)
- [PDF Link](https://arxiv.org/pdf/2511.16282v2.pdf)

## Subfields
 语义SLAM / 3D场景理解 (Semantic SLAM / 3D Scene Understanding)
## Reason for Interest

1. 领域相关性（低）：该论文明确聚焦于“室内环境”和“辅助导航”（assistive navigation），使用手持设备视角，针对的是服务机器人或视障辅助场景，而非车端自动驾驶。根据评分规则，非车端应用最高5分。
2. 创新性：提出了分块（Block-wise）处理VGGT以降低显存需求并支持长序列流式SLAM，结合YOLOv9e进行语义实例一致性追踪，具有一定的工程价值。
3. 性能表现：虽然实现了流式处理和低显存占用，但在通过TUM和7-Scenes数据集的评估中，其绝对轨迹误差（ATE）和重建精度均略低于非流式的VGGT-SLAM基线，未达成SOTA。
4. 结论：属于计算机视觉与机器人领域的扎实工作，但对高速、室外、动态复杂的自动驾驶核心场景直接参考价值有限。
