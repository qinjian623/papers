# [9.2] MambaTrack3D: A State Space Model Framework for LiDAR-Based Object Tracking under High Temporal Variation

- Authors: Shengjing Tian, Yinan Han, Xiantong Zhao, Xuehu Liu, Qi Lang
- [arXiv Link](https://arxiv.org/abs/2511.15077v1)
- [PDF Link](https://arxiv.org/pdf/2511.15077v1.pdf)

## Subfields
 3D单目标跟踪 (LiDAR Single Object Tracking)
## Reason for Interest

1. **创新性强**：该论文紧跟Mamba（状态空间模型）架构的前沿趋势，首次将其成功应用于LiDAR 3D跟踪领域，提出了MIP（帧间传播）和GFEM（分组特征增强）模块，有效解决了Transformer架构计算复杂度随点数二次增长的问题。
2. **行业价值高**：针对“高时间变化（HTV）”场景（即传感器丢帧或低频处理）的优化具有极高的工程落地价值，能显著降低对边缘计算设备的算力要求（实现线性复杂度），非常契合车载部署需求。
3. **结果可信**：实验设计严谨，在KITTI和nuScenes构建的HTV数据集上进行了详尽的对比，性能提升显著（尤其是在难例nuScenes上），且消融实验充分证明了各模块的有效性。
4. **综合评价**：兼顾了高精度与高效率，是一个优秀的自动驾驶感知算法工作。
