# [8.2] Lumos: Let there be Language Model System Certification

- Authors: Isha Chaudhary, Vedaant Jain, Avaljot Singh, Kavya Sachdeva, Sayan Ranu, Gagandeep Singh
- [arXiv Link](https://arxiv.org/abs/2512.02966v1)
- [PDF Link](https://arxiv.org/pdf/2512.02966v1.pdf)

## Subfields
 自动驾驶安全验证 / VLM / 仿真测试
## Reason for Interest

论文针对大模型（特别是 VLM）在自动驾驶等安全关键领域的应用，提出了一种基于图的概率编程语言 LUMOS 进行形式化认证。创新性在于通过领域特定语言（DSL）将高层抽象场景与底层仿真器（Scenic/CARLA）连接，能够系统地生成分布外或对抗性场景（如恶劣天气）来评估模型边界。虽然论文核心贡献在于编程语言设计，但其实验部分直接针对自动驾驶 VLM 的安全性问题，提供了量化的鲁棒性评估方法，对于大模型上车的安全验证与测试具有重要的指导意义和工具价值。
