# [9.2] MiMo-Embodied: X-Embodied Foundation Model Technical Report

- Authors: Xiaoshuai Hao, Lei Zhou, Zhijian Huang, Zhiwen Hou, Yingbo Tang, Lingfeng Zhang, Guang Li...
- [arXiv Link](https://arxiv.org/abs/2511.16518v1)
- [PDF Link](https://arxiv.org/pdf/2511.16518v1.pdf)

## Subfields
 端到端自动驾驶 / 多模态大模型 (VLM) / 具身智能
## Reason for Interest

该论文提出了MiMo-Embodied，一个融合了自动驾驶与具身智能的通用多模态大模型。其核心价值在于：1. **方法创新**：验证了具身智能任务（如机械臂操作）与自动驾驶任务之间的正向迁移能力，提出了包含CoT（思维链）和RL（强化学习，特别是GRPO算法）的四阶段训练策略，显著增强了模型的推理和规划能力。2. **实验充分**：在12个自动驾驶基准和17个具身智能基准上进行了全面评估，覆盖感知、预测、规划全栈任务，数据详实。3. **行业前瞻性**：将DeepSeek-R1式的强化学习（GRPO）应用于自动驾驶规划，并开源了代码和模型，对端到端自动驾驶研究具有极高的参考价值。
