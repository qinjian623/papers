# [9.0] Beyond Frame-wise Tracking: A Trajectory-based Paradigm for Efficient Point Cloud Tracking

- Authors: BaiChen Fan, Sifan Zhou, Jian Li, Shibo Zhao, Muqing Cao, Qin Wang
- [arXiv Link](https://arxiv.org/abs/2509.11453v1)
- [PDF Link](https://arxiv.org/pdf/2509.11453v1.pdf)

## Subfields
 3D单目标跟踪 (3D Single Object Tracking) / 激光雷达感知
## Reason for Interest

1. 创新性：针对点云跟踪中‘双帧方法缺乏长期上下文’与‘序列方法计算昂贵’的矛盾，提出了一种巧妙的‘轨迹为基（Trajectory-based）’范式。核心创新在于Implicit Motion Modeling (IMM) 模块，该模块仅利用轻量级的历史包围盒（BBox）序列通过Transformer学习长期运动规律，而不增加繁重的点云序列处理开销，设计优雅且高效。
2. 实验效果：在权威的大规模nuScenes数据集上取得了显著的SOTA性能，特别是在精确度上有大幅提升（+4.48%），证明了引入长期运动先验的有效性。同时，消融实验展示了该模块在不同基线（BAT, M2-Track）上的通用性。
3. 行业价值：该方法在大幅提升精度的同时，仍保持了56 FPS的实时运行速度（RTX 3090），具有极高的车端部署潜力和实用价值。
4. 完整性：论文逻辑清晰，实验对比充分（涵盖不同类别、稀疏度分析、速度对比），并提供了详细的可视化分析支持结论。
