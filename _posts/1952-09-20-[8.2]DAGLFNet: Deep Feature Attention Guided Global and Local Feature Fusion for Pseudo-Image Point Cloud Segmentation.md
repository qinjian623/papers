# [8.2] DAGLFNet: Deep Feature Attention Guided Global and Local Feature Fusion for Pseudo-Image Point Cloud Segmentation

- Authors: Chuang Chen, Yi Lin, Bo Wang, Jing Hu, Xi Wu, Wenyi Ge
- [arXiv Link](https://arxiv.org/abs/2510.10471v2)
- [PDF Link](https://arxiv.org/pdf/2510.10471v2.pdf)

## Subfields
 LiDAR 点云语义分割 / Range-View 方法
## Reason for Interest

论文提出了一种名为 DAGLFNet 的伪图像（Range View）基点云分割网络，针对性地解决了投影过程中的特征模糊和深度冲突问题。1. 创新性：提出了 GL-FFE（全局-局部特征融合）、MB-FE（多分支特征提取）和 FFDFA（深度引导注意力融合）三个模块，设计逻辑清晰，有效提升了细粒度特征的表达能力。2. 实验完整性：在 SemanticKITTI 和 nuScenes 两大主流数据集上进行了详细对比，消融实验充分，且包含速度与精度的权衡分析（45.4 ms/scan，实时性好）。3. 结果可信度：在验证集上指标优于当前主流方法（如 WaffleIron, FARVNet），展示了较强的竞争力。4. 扣分项：评估仅基于验证集（Validation Set），缺乏官方测试集（Test Set/Leaderboard）的最终跑分结果，SOTA 结论的说服力略打折扣。
