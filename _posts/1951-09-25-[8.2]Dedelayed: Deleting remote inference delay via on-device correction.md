# [8.2] Dedelayed: Deleting remote inference delay via on-device correction

- Authors: Dan Jacobellis, Mateen Ulhaq, Fabien Racapé, Hyomin Choi, Neeraja J. Yadwadkar
- [arXiv Link](https://arxiv.org/abs/2510.13714v2)
- [PDF Link](https://arxiv.org/pdf/2510.13714v2.pdf)

## Subfields
 云边协同感知 (Cloud-Edge Collaborative Perception) / 语义分割
## Reason for Interest

1. 创新性强：针对车路协同/云端增强感知中的核心痛点——网络延迟，提出了一种“云端预测未来特征+端侧当前特征融合”的新颖架构，有效消除了通信延迟导致的预测滞后问题。
2. 方案合理：利用云端大模型的时序预测能力补偿传输耗时，同时保留端侧小模型的实时性作为安全兜底，符合自动驾驶对安全冗余的要求。
3. 实验设计：在BDD100K驾驶数据集上进行了详细的延迟消融实验（0-167ms），验证了方法在不同网络条件下的鲁棒性。
4. 局限性：评估依赖于DepthAnything生成的伪标签，而非人工标注真值，且主要对比的是自构建的基线模型（Local/Remote/Split），缺乏与其他先进云边协同算法（如FCM等）的直接定量对比，因此未认定为标准榜单SOTA。但对V2X和软件定义汽车（SDV）领域的感知架构设计具有较高参考价值。
