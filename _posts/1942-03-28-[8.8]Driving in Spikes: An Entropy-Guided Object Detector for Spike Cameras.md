# [8.8] Driving in Spikes: An Entropy-Guided Object Detector for Spike Cameras

- Authors: Ziyan Liu, Qi Su, Lulu Tang, Zhaofei Yu, Tiejun Huang
- [arXiv Link](https://arxiv.org/abs/2511.15459v1)
- [PDF Link](https://arxiv.org/pdf/2511.15459v1.pdf)

## Subfields
 Neuromorphic Perception / Object Detection
## Reason for Interest

该论文针对自动驾驶中高速运动模糊和极端光照痛点，探索了 Spike 相机（脉冲相机）在目标检测中的应用，具有极高的行业前瞻价值。主要贡献包括：1. 提出了 EASD 双分支网络架构，结合时序纹理模块和熵选择性注意力模块，有效解决了 Spike 数据稀疏且缺乏纹理的问题；2. 构建了首个面向自动驾驶的 Spike 检测仿真基准 DSEC-Spike，填补了数据空白；3. 实验在仿真和真实数据上均取得了显著的 SOTA 性能，证明了该技术路线在高速、高动态场景下的鲁棒性。工作完整度高，创新性强。
