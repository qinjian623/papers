# [6.0] Robust 3DGS-based SLAM via Adaptive Kernel Smoothing

- Authors: Shouhe Zhang, Dayong Ren, Sensen Song, Wenjie Li, Piaopiao Yu, Yurong Qian
- [arXiv Link](https://arxiv.org/abs/2511.23221v1)
- [PDF Link](https://arxiv.org/pdf/2511.23221v1.pdf)

## Subfields
 3D Mapping / SLAM (3D Gaussian Splatting)
## Reason for Interest

论文提出了基于自适应核平滑（CB-KNN）的 3DGS SLAM 优化方法，有效缓解了高斯参数噪声对位姿追踪的影响，技术思路具有较好的创新性和合理性，且在室内 RGB-D 数据集上超越了 SplaTAM 等基线。然而，作为自动驾驶领域的评审，该工作存在明显局限：1. 实验完全基于室内小场景 RGB-D 数据（Replica, TUM, ScanNet），缺乏 KITTI、nuScenes 等大规模室外自动驾驶场景的验证；2. 强依赖稠密深度图输入，未针对车端常用的稀疏激光雷达或纯视觉方案进行适配。尽管 3DGS 是自动驾驶高精地图重建的热点，但该论文目前的形态更偏向室内机器人，对车端感知的直接参考价值有限，故给予及格分但不过高。
