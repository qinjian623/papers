# [8.2] CARScenes: Semantic VLM Dataset for Safe Autonomous Driving

- Authors: Yuankai He, Weisong Shi
- [arXiv Link](https://arxiv.org/abs/2511.10701v2)
- [PDF Link](https://arxiv.org/pdf/2511.10701v2.pdf)

## Subfields
 自动驾驶数据集 / VLM场景理解 / 自动驾驶安全
## Reason for Interest

1. 创新性与价值（8.5/10）：论文敏锐地指出了当前自动驾驶数据集偏向底层感知（Box/Mask），缺乏高层语义（如路况、行为逻辑、风险等级）的问题。提出的 CARScenes 数据集通过结构化 Schema 和风险评分（Severity），为自动驾驶 VLM 的解释性和安全性研究提供了急需的数据支撑。
2. 方法论（8.0/10）：采用 'VLM预标注 + 确定性校验 + 人工审核' 的流水线在保证质量的同时降低了成本，方法科学且可复现。
3. 实验与完整性（7.5/10）：提供了详尽的 Schema 定义、工具链和 Baseline 模型。唯一的短板在于数据规模（5,192帧）相对较小，虽然对评测和微调足够，但相比大规模预训练数据集仍显单薄。
4. 行业潜力（9.0/10）：该数据集对于推动端到端大模型在长尾场景理解、风险评估方面的能力具有很高的实用价值，且 Schema 设计对行业数据治理有参考意义。
