# [4.5] Learning Neural Control Barrier Functions from Expert Demonstrations using Inverse Constraint Learning

- Authors: Yuxuan Yang, Hussein Sibai
- [arXiv Link](https://arxiv.org/abs/2510.21560v1)
- [PDF Link](https://arxiv.org/pdf/2510.21560v1.pdf)

## Subfields
 规划控制 / 安全保障 (Planning & Control / Safety Assurance)
## Reason for Interest

该论文提出了利用逆约束学习（ICL）从专家演示中学习神经控制障碍函数（Neural CBF）的方法，在理论上具有一定的创新性，能够解决安全约束难以显式定义的问题。然而，在自动驾驶领域的应用价值评估中，该论文存在显著短板：1. 实验环境过于简化，使用的Dubins Car模型和静态障碍物场景（如简单的圆形/方形避障）远低于自动驾驶实际场景的复杂性（缺乏动态交通流、复杂道路结构、感知噪声等）；2. 缺乏在主流高保真自动驾驶仿真器（如CARLA, nuPlan）或真实数据集上的验证，难以证明其在现代自动驾驶协议栈中的有效性；3. 虽然研究动机提及自动驾驶（如跟车场景），但实际验证更偏向于通用机器人控制理论。因此，尽管方法论有趣，但距离车端实际落地和行业SOTA水平有较大差距，评分受限。
