# [8.8] Learning from Mistakes: Loss-Aware Memory Enhanced Continual Learning for LiDAR Place Recognition

- Authors: Xufei Wang, Junqiao Zhao, Siyue Tao, Qiwen Gu, Wonbong Kim, Tiantian Feng
- [arXiv Link](https://arxiv.org/abs/2511.15597v1)
- [PDF Link](https://arxiv.org/pdf/2511.15597v1.pdf)

## Subfields
 Localization / SLAM (LiDAR Place Recognition)
## Reason for Interest

论文针对自动驾驶在动态环境中长期运行面临的灾难性遗忘问题，提出了一种基于损失感知采样和复习增强的持续学习框架(KDF+)。
1. 创新性：通过预测样本Loss来评估学习难度并进行加权采样的策略设计巧妙，配合复习增强损失(Rehearsal Enhancement Loss)有效巩固了旧知识。
2. 实验完整性：在三个主流LPR骨干网络(PointNetVLAD, MinkLoc3D, CASSPR)和四个不同环境数据集上进行了详尽的测试，包含消融实验和泛化性验证(应用到其他CL方法)。
3. 结果可信度：实验数据显示在保持旧环境性能方面有显著优势，遗忘率极低，且方法具有良好的通用性。
4. 行业价值：对于构建能够适应长期环境变化的自动驾驶定位系统具有较高的实用价值。
