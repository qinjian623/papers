# [8.5] UNCAP: Uncertainty-Guided Planning Using Natural Language Communication for Cooperative Autonomous Vehicles

- Authors: Neel P. Bhatt, Po-han Li, Kushagra Gupta, Rohan Siva, Daniel Milan, Alexander T. Hogue, S...
- [arXiv Link](https://arxiv.org/abs/2510.12992v1)
- [PDF Link](https://arxiv.org/pdf/2510.12992v1.pdf)

## Subfields
 V2X 协同规划 / 大模型 (VLM) 决策
## Reason for Interest

论文提出了一种结合不确定性量化（Conformal Prediction）与自然语言通信的 V2X 协同规划框架 UNCAP。其核心创新在于设计了 BARE 和 SPARE 两阶段通信筛选机制，有效解决了多车通信中的带宽瓶颈与无关信息干扰问题。实验设计合理，在 CARLA/OPV2V 数据集上证明了其在低带宽下仍能大幅提升规划安全性。尽管依赖仿真且 VLM 推理延迟（约 1.3s）较高，但其针对高层决策的异步处理逻辑具有可行性，为大模型在 V2X 中的应用提供了新思路。
