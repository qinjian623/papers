# [4.5] MT-Depth: Multi-task Instance feature analysis for the Depth Completion

- Authors: Abdul Haseeb Nizamani, Dandi Zhou, Xinhai Sun
- [arXiv Link](https://arxiv.org/abs/2512.04734v2)
- [PDF Link](https://arxiv.org/pdf/2512.04734v2.pdf)

## Subfields
 3D感知 / 深度补全 (Depth Completion)
## Reason for Interest

该论文提出利用冻结的YOLOv11实例分割掩码通过交叉注意力机制引导深度补全，思路具有一定的合理性，旨在优化物体边缘的深度质量。然而，作为一篇面向自动驾驶的论文，其存在严重缺陷：1. **实验验证极弱**：仅在合成数据集（Virtual KITTI 2）上进行验证，完全缺失行业通用的真实场景数据集（如KITTI Depth, Waymo, nuScenes）测试，结果缺乏实际应用可信度。2. **基线对比不足**：仅对比了基础U-Net和较老的SemSegDepth，未对比NLSPN、GuideNet、DySPN等主流SOTA方法，无法证明其在当前技术栈中的竞争力。3. **指标矛盾**：虽然RMSE显著降低，但MAE（平均绝对误差）相比基础U-Net反而上升（61.1 -> 65.6），说明其虽改善了部分大误差区域（如边缘），但在整体像素级的准确性上可能存在退化。4. **实际价值低**：依赖合成数据的‘SOTA’在自动驾驶实车部署中参考价值有限。
