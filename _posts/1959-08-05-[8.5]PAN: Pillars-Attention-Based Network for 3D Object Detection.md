# [8.5] PAN: Pillars-Attention-Based Network for 3D Object Detection

- Authors: Ruan Bispo, Dane Mitrev, Letizia Mariotti, Clément Botty, Denver Humphrey, Anthony Scanla...
- [arXiv Link](https://arxiv.org/abs/2509.15935v2)
- [PDF Link](https://arxiv.org/pdf/2509.15935v2.pdf)

## Subfields
 3D Perception / Camera-Radar Fusion
## Reason for Interest

该论文提出了一种名为 PAN (Pillars-Attention-Based Network) 的新型架构，专注于高效的 Camera-Radar 融合。主要创新点在于设计了结合 Pillars 和 Self-Attention 的雷达骨干网络，有效处理了雷达数据的稀疏性，并替代了耗时的 FPN 结构以提升推理速度。实验结果表明，该方法在 nuScenes 数据集上（特别是 ResNet-50 设定下）在保持实时性（~30 FPS）的同时，NDS 指标超越了之前的 SOTA 方法（CRN），且在雨天和夜间等恶劣天气下表现出显著的鲁棒性。虽然 mAP 略低，但通过雷达改善了速度和方向估计（提升了 NDS）。
