# [8.8] WinMamba: Multi-Scale Shifted Windows in State Space Model for 3D Object Detection

- Authors: Longhui Zheng, Qiming Xia, Xiaolu Chen, Zhaoliang Liu, Chenglu Wen
- [arXiv Link](https://arxiv.org/abs/2511.13138v1)
- [PDF Link](https://arxiv.org/pdf/2511.13138v1.pdf)

## Subfields
 3D感知 / LiDAR目标检测 (State Space Models)
## Reason for Interest

该论文针对Mamba架构在3D点云检测中因固定窗口划分导致的空间特征不连续问题，提出了多尺度移动窗口（WSF）和自适应窗口融合（AWF）策略。这种设计成功地结合了Mamba的线性计算复杂度优势与更精细的空间上下文建模能力。实验结果表明，该方法在KITTI数据集上取得了显著的性能提升（mAP提升约2.9%），在Waymo数据集上也验证了其有效性（虽然因算力限制仅使用20%数据，但对比基线仍有优势）。考虑到Mamba架构在端侧推理效率上的巨大潜力，该工作对于自动驾驶车载计算平台具有极高的应用价值和启发意义。
