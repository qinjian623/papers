# [7.8] Driving scenario generation and evaluation using a structured layer representation and foundational models

- Authors: Arthur Hubert, Gamal Elghazaly, Rapha\"el Frank
- [arXiv Link](https://arxiv.org/abs/2511.01541)
- [PDF Link](https://arxiv.org/pdf/2511.01541.pdf)

## Subfields
 自动驾驶场景生成与评估 / 大模型在自动驾驶数据增强中的应用
## Reason for Interest

该论文提出了一种基于结构化分层表示和基础模型（LLM/MLLM）的自动驾驶场景生成与评估框架，其核心贡献和价值体现在以下几个方面：

1.  **创新性 (8/10)**: 论文提出了一种新颖的五层结构化模型来表示自动驾驶场景，这种模型将场景分解为道路结构、路边结构、临时变化、动态物体和环境条件等不同层级，极大地提升了场景描述的精细度和可控性。利用多模态大语言模型（MLLM）从真实数据（nuScenes）中提取结构化场景，并通过引导LLM对特定层级进行编辑来生成稀有或边缘场景，这种结合LLM生成能力与结构化指导的方法具有较强的创新性。此外，论文还对现有基于语义嵌入的场景评估指标（多样性、原创性/OODness）进行了改进和扩展，使其能适用于分层结构化场景的评估，并支持层级分析。

2.  **实验完整性 (7.5/10)**: 实验基于nuScenes-mini数据集（包含10个20秒的场景），系统性地比较了非结构化、软结构化和硬结构化五层模型在生成场景时的表现，并对比了不同的LLM提示策略对生成质量的影响。通过引入改进的评估指标，对生成的场景在不同层级上的多样性和原创性进行了定量分析。虽然实验数据集规模相对较小，但其设计足以验证所提出框架和核心思想的有效性及不同组件的影响。论文还使用世界基座模型（Veo3）对生成的文本场景进行了视频生成演示，尽管视频质量存在一些不连贯性，但作者坦诚地讨论了这些限制。

3.  **可信度 (8/10)**: 论文使用了Google Gemini 2.5 Pro作为MLLM和LLM，以及Gemini-embedding-001作为语义嵌入模型，这些都是业界领先且性能强大的基础模型，增加了研究的可靠性。作者对生成视频中存在的局限性（例如，LLM可能去除或分组原始场景中的物体，以及视频生成中的不连贯和幻觉）进行了公开讨论，展现了严谨的科学态度。

4.  **行业潜力 (8.5/10)**: 稀有和边缘场景的生成对于自动驾驶系统的测试、验证和安全性至关重要。该论文提出的结构化场景生成与评估框架，为自动驾驶系统提供了一种可控、可解释、且能够针对性生成关键场景的强大工具。这种方法有助于解决现有数据集中边缘案例不足的问题，并能加速自动驾驶系统的开发和部署。结构化表示也为未来实现更精确的场景编辑和人机协作提供了良好的基础。

**总评**: 尽管论文主要聚焦于自动驾驶的“场景生成与评估”这一辅助性但极其重要的研究方向，而非车端直接运行的感知、决策或控制模块，但其对自动驾驶安全性、鲁棒性验证和数据增强的推动作用是基础且战略性的。论文在创新性、方法论的严谨性和未来潜力方面表现出色。目前从文本描述到高保真视频生成的挑战依然存在，但论文的核心贡献在于其结构化文本场景的生成与评估框架，这在自动驾驶领域具有显著价值。因此，该研究值得一个较高的评分。
## Abstract: 
Rare and challenging driving scenarios are critical for autonomous vehicle development. Since they are difficult to encounter, simulating or generating them using generative models is a popular approach. Following previous efforts to structure driving scenario representations in a layer model, we propose a structured five-layer model to improve the evaluation and generation of rare scenarios. We use this model alongside large foundational models to generate new driving scenarios using a data augmentation strategy. Unlike previous representations, our structure introduces subclasses and characteristics for every agent of the scenario, allowing us to compare them using an embedding specific to our layer-model. We study and adapt two metrics to evaluate the relevance of a synthetic dataset in the context of a structured representation: the diversity score estimates how different the scenarios of a dataset are from one another, while the originality score calculates how similar a synthetic dataset is from a real reference set. This paper showcases both metrics in different generation setup, as well as a qualitative evaluation of synthetic videos generated from structured scenario descriptions. The code and extended results can be found at https://github.com/Valgiz/5LMSG.
