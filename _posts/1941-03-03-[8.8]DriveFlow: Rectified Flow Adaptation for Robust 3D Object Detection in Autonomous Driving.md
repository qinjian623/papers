# [8.8] DriveFlow: Rectified Flow Adaptation for Robust 3D Object Detection in Autonomous Driving

- Authors: Hongbin Lin, Yiming Yang, Chaoda Zheng, Yifan Zhang, Shuaicheng Niu, Zilu Guo, Yafeng Li,...
- [arXiv Link](https://arxiv.org/abs/2511.18713v1)
- [PDF Link](https://arxiv.org/pdf/2511.18713v1.pdf)

## Subfields
 自动驾驶感知 / 生成式数据增强 / 鲁棒性研究
## Reason for Interest

1. 创新性：论文首次将Rectified Flow（如Stable Diffusion 3, FLUX）引入自动驾驶数据增强，提出基于频率分解的编辑策略，有效解决了生成式AI在保留物体3D几何形状与风格迁移之间的矛盾（High-Frequency Foreground Preservation）。
2. 实用价值：相比传统的Diffusion Inversion方法，DriveFlow实现了数量级的速度提升（秒级 vs 分钟级），使得大规模合成训练数据成为可能，具有极高的行业落地潜力。
3. 实验完整性：在单目（MonoFlex）、多视角（BEVFormer）、时序（BEVDet4D）等多种检测器架构上验证了有效性，且在KITTI-C和nuScenes-C等OOD基准上取得了SOTA的鲁棒性提升。
4. 可信度：提供了详细的消融实验、可视化对比及代码开源链接，结论扎实。
