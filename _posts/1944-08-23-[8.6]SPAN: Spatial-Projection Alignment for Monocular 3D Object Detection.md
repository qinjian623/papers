# [8.6] SPAN: Spatial-Projection Alignment for Monocular 3D Object Detection

- Authors: Yifan Wang, Yian Zhao, Fanqi Pu, Xiaochen Yang, Yang Tang, Xi Chen, Wenming Yang
- [arXiv Link](https://arxiv.org/abs/2511.06702v1)
- [PDF Link](https://arxiv.org/pdf/2511.06702v1.pdf)

## Subfields
 单目3D目标检测 (Monocular 3D Object Detection)
## Reason for Interest

该论文针对单目3D检测中属性回归解耦导致的几何不一致问题，提出了SPAN框架（Spatial-Projection Alignment）。

1. **创新性 (8/10)**：提出了“空间点对齐”和“3D-2D投影对齐”两个显式几何约束模块，并设计了“分层任务学习”策略（Hierarchical Task Learning）来解决引入强几何约束导致的早期训练不稳定问题。这种将几何先验与端到端学习结合且作为即插即用模块的设计具有较好的创新性。
2. **实验完整性 (9/10)**：在KITTI和Waymo两个权威数据集上进行了广泛验证。不仅在自己的基线上测试，还将其集成到MonoDETR、MoVis等不同架构中证明了通用性。包含详细的消融实验、噪声鲁棒性分析和定性可视化，实验工作非常扎实。
3. **可信度 (9/10)**：理论推导清晰（如附录中对投影约束的凸性证明），SOTA比较公平（区分了使用额外数据与否），代码实现细节披露较多（如损失权重调度公式）。
4. **行业潜力 (8/10)**：单目3D检测是低成本自动驾驶方案的关键，该方法作为通用插件能以零推理成本提升性能，具有较高的落地应用价值。

综合来看，这是一篇在单目3D感知领域质量很高的论文。
