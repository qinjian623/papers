# [4.5] PCR-ORB: Enhanced ORB-SLAM3 with Point Cloud Refinement Using Deep Learning-Based Dynamic Object Filtering

- Authors: Sheng-Kai Chen, Jie-Yu Chao, Jr-Yu Chang, Po-Lien Wu, Po-Chiang Lin
- [arXiv Link](https://arxiv.org/abs/2512.23318v1)
- [PDF Link](https://arxiv.org/pdf/2512.23318v1.pdf)

## Subfields
 Visual SLAM / 动态环境感知 (Dynamic SLAM)
## Reason for Interest

该论文提出将 YOLOv8 语义分割与 ORB-SLAM3 结合以过滤动态物体，并利用 CUDA 加速。评分较低的主要原因如下：
1. **创新性有限**：利用语义分割掩膜剔除特征点是动态 SLAM 领域非常成熟且常规的技术路线（类似 DynaSLAM），本文仅更新了检测器（YOLOv8）并增加了常规的几何/运动约束，缺乏深度的理论创新。
2. **实验结果不稳定**：论文诚实地展示了混合结果——虽然在典型动态场景（如 Seq 04）有提升，但在 Seq 05、06、08 等场景中精度反而下降，表明该策略存在“过度过滤”或误删静态特征的风险，鲁棒性不足。
3. **对比基线薄弱**：仅对比了原始 ORB-SLAM3，缺乏与现有的 SOTA 动态 SLAM 算法的横向对比，无法证明其在动态 SLAM 细分领域的竞争力。
4. **工程属性重于研究**：主要贡献在于 CUDA 工程实现与集成，而非算法层面的突破。
