# [8.5] Have We Scene It All? Scene Graph-Aware Deep Point Cloud Compression

- Authors: Nikolaos Stathoulopoulos, Christoforos Kanellakis, George Nikolakopoulos
- [arXiv Link](https://arxiv.org/abs/2510.08512v2)
- [PDF Link](https://arxiv.org/pdf/2510.08512v2.pdf)

## Subfields
 3D点云压缩 / 车路协同 (V2X) / 多车协同感知
## Reason for Interest

1. 创新性强：提出利用语义场景图（Scene Graph）指导点云压缩，将几何信息与语义结构解耦，既保留了关键的感知特征，又大幅降低了带宽需求。
2. 实验完整：在主流自动驾驶数据集（SemanticKITTI, nuScenes）上进行了详尽的对比实验，不仅评估了压缩指标，还验证了下游机器人任务（多车位姿图优化 PGO、地图合并）的性能，证明了其在V2X带宽受限场景下的实用价值。
3. 泛化能力：展示了模型在未微调的情况下从SemanticKITTI泛化到nuScenes的能力，优于过拟合局部几何的对比方法。
4. 行业潜力：尽管目前推理速度（约2.5Hz）距离实时激光雷达帧率（10Hz+）仍有差距，但其对语义信息的显式编码使其在众包地图更新、多车协同等不需要全帧率但需要高语义一致性的场景中极具潜力。
