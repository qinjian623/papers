# [4.5] MS-YOLO: Infrared Object Detection for Edge Deployment via MobileNetV4 and SlideLoss

- Authors: Jiali Zhang, Thomas S. White, Haoliang Zhang, Wenqing Hu, Donald C. Wunsch, Jian Liu
- [arXiv Link](https://arxiv.org/abs/2509.21696v1)
- [PDF Link](https://arxiv.org/pdf/2509.21696v1.pdf)

## Subfields
 红外感知 / 边缘侧目标检测 (Infrared Perception / Edge Object Detection)
## Reason for Interest

1. **创新性低**：论文主要工作是将YOLOv8的CSPDarknet主干替换为MobileNetV4，并应用已有的SlideLoss损失函数，属于标准模块组合的工程应用，缺乏理论深度或架构创新。
2. **实验结果负面**：根据文中Table III数据，作者提出的MS-YOLO模型相比其训练的基线（YOLOv8n*），在关键的mAP和Recall指标上均出现显著退步。对于自动驾驶场景，尤其是存在遮挡的红外检测，Recall（召回率）的降低是致命的（易导致漏检），而论文仅强调了Precision的提升。
3. **权衡价值存疑**：以牺牲大量检测性能（mAP -6.3%相对值）来换取微乎其微的算力节省（0.1 GFLOPs），在实际工程中并非有效的优化方向。
4. **结论不可信**：论文试图通过强调Precision和极小的计算量减少来包装结果，掩盖了模型综合检测能力下降的事实。
