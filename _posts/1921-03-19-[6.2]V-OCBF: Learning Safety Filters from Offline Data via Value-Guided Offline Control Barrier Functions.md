# [6.2] V-OCBF: Learning Safety Filters from Offline Data via Value-Guided Offline Control Barrier Functions

- Authors: Mumuksh Tayal, Manan Tayal, Aditya Singh, Shishir Kolathaya, Ravi Prakash
- [arXiv Link](https://arxiv.org/abs/2512.10822v1)
- [PDF Link](https://arxiv.org/pdf/2512.10822v1.pdf)

## Subfields
 规划控制 / 离线安全强化学习 (Planning & Control / Offline Safe RL)
## Reason for Interest

论文提出了一种结合Expectile Regression的离线控制障碍函数（CBF）学习框架（V-OCBF），有效解决了离线强化学习中因查询分布外（OOD）动作导致的安全估计偏差问题，具有较好的理论创新性。方法在标准的RL安全基准（Safety Gymnasium）和简化的车辆避障场景（AGV）中表现优异。然而，评分受限于实验环境的简易性：AGV实验仅基于简单的Dubins Car模型，未涉及复杂的车辆动力学或逼真的自动驾驶交通场景（如CARLA、nuScenes闭环等），且推断阶段仍依赖于学习到的动力学模型，这在实际自动驾驶的高风险场景中可能存在鲁棒性问题。因此，虽然方法论有价值，但对自动驾驶行业的直接落地指导意义有限。
