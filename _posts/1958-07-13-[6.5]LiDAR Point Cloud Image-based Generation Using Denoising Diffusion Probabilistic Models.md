# [6.5] LiDAR Point Cloud Image-based Generation Using Denoising Diffusion Probabilistic Models

- Authors: Amirhesam Aghanouri, Cristina Olaverri-Monreal
- [arXiv Link](https://arxiv.org/abs/2509.18917v1)
- [PDF Link](https://arxiv.org/pdf/2509.18917v1.pdf)

## Subfields
 自动驾驶仿真 / 激光雷达点云生成 (LiDAR Generation)
## Reason for Interest

论文将DDPM应用于LiDAR点云生成，并提出了新的噪声调度（Time-dependent/Ramp）和傅里叶时间步嵌入方法。优点在于：1. 针对点云生成的扩散过程进行了细致的消融实验；2. 验证了IAMCV数据集；3. 强调了模型的训练资源需求较低（单卡A6000可训），相比LiDMs（需8卡）更具复现性。缺点非常明显：1. 核心性能指标未能超越现有SOTA（LiDMs）；2. 推理速度（~1.68s）慢于LiDMs（~1.35s）；3. 虽然声称轻量化，但在效率和质量双重维度上均未取得优势，属于对现有方法的增量式改进而非突破。
