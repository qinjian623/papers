# [8.5] Vision-Based Perception for Autonomous Vehicles in Off-Road Environment Using Deep Learning

- Authors: Nelson Alves Ferreira Neto
- [arXiv Link](https://arxiv.org/abs/2509.19378v1)
- [PDF Link](https://arxiv.org/pdf/2509.19378v1.pdf)

## Subfields
 Visual Perception / Semantic Segmentation (Off-road & Adverse Conditions)
## Reason for Interest

该论文（博士学位论文）在非结构化道路（Off-road）及恶劣天气（夜间、雨天、扬尘）下的视觉感知领域做出了扎实贡献。创新性体现在提出了CMSNet模块化框架，针对嵌入式端侧计算资源进行了优化，平衡了精度与实时性；填补了自动驾驶在非铺装路面数据集的空白（发布Kamino数据集）。实验非常完整，涵盖了从模型消融、与SOTA对比（DeepScene数据集）、恶劣环境鲁棒性分析到真车实地部署测试的全流程。虽然核心架构组件（如MobileNetV2, ASPP）基于现有技术组合，但其在特定垂直领域的工程化落地与系统性评估具有很高的行业参考价值。
