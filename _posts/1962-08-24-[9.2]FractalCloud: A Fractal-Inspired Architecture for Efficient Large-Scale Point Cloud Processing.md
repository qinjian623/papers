# [9.2] FractalCloud: A Fractal-Inspired Architecture for Efficient Large-Scale Point Cloud Processing

- Authors: Yuzhe Fu, Changchun Zhou, Hancheng Ye, Bowen Duan, Qiyu Huang, Chiyue Wei, Cong Guo, Hai ...
- [arXiv Link](https://arxiv.org/abs/2511.07665v2)
- [PDF Link](https://arxiv.org/pdf/2511.07665v2.pdf)

## Subfields
 自动驾驶计算芯片 / 3D点云感知加速
## Reason for Interest

论文针对自动驾驶中大规模LiDAR点云处理的‘访存墙’问题，提出了一种极具创新性的软硬协同设计（FractalCloud）。

1. 创新性（高）：提出了基于分形几何的形状感知划分方法（Fractal Partitioning）和块级并行点操作（BPPO），成功将O(n^2)的全局搜索复杂度降维，解决了PointNet++等网络在大规模场景下‘点操作’（采样、搜索）的效率瓶颈。
2. 实验完整性（极高）：基于28nm工艺综合评估，对比了GPU及多个前沿加速器（Mesorasi, PointAcc, Crescent），在不同模型（PointNeXt, PointVector）和不同规模数据上进行了详尽的消融实验。
3. 行业价值（高）：该方案显著提升了车载边缘端处理大规模点云（如300K+点/帧）的实时性与能效，对自动驾驶感知系统的落地具有重要参考价值。
