# [6.5] SnowyLane: Robust Lane Detection on Snow-covered Rural Roads Using Infrastructural Elements

- Authors: Jörg Gamerdinger, Benedict Wetzel, Patrick Schulz, Sven Teufel, Oliver Bringmann
- [arXiv Link](https://arxiv.org/abs/2511.05108v1)
- [PDF Link](https://arxiv.org/pdf/2511.05108v1.pdf)

## Subfields
 感知 (Perception) / 车道线检测 (Lane Detection) / 恶劣天气鲁棒性
## Reason for Interest

论文针对自动驾驶中的长尾难题——积雪覆盖导致车道线不可见，提出了一种利用道路基础设施（反光柱/Delineators）进行车道线几何重建的方法，并发布了合成数据集SnowyLane。

1. 创新性与价值（7分）：思路具有实用价值，利用乡村道路常见的反光柱作为视觉/雷达锚点，通过贝塞尔曲线拟合车道，规避了直接检测积雪覆盖车道线的困难。发布的包含多级积雪强度的合成数据集对研究恶劣天气感知有一定贡献。

2. 实验完整性（7分）：对比了直接检测方法（SCNN, RESA等）与本文提出的间接方法，涵盖了Camera（Faster R-CNN, YOLO等）和LiDAR（PV-RCNN等）多种感知模型，消融实验和实时性分析较为充分。

3. 可信度与局限性（5分）：这是论文最大的扣分点。所有实验完全基于CARLA仿真生成的合成数据。积雪环境下的视觉特征（纹理、反光）和雷达噪声在真实世界中远比仿真复杂，缺乏实车数据验证使得该方法在真实场景下的有效性（Sim-to-Real gap）存疑。此外，该方法强依赖于特定道路设施（反光柱），在基础设施不同的国家或地区无法通用。

综上，该论文提供了一个特定ODD（设计运行域）下的有效工程方案和仿真基准，但受限于纯仿真验证和场景依赖性，行业通用推广潜力受限。
