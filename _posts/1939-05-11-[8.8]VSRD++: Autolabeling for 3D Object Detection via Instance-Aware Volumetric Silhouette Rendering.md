# [8.8] VSRD++: Autolabeling for 3D Object Detection via Instance-Aware Volumetric Silhouette Rendering

- Authors: Zihua Liu, Hiroki Sakuma, Masatoshi Okutomi
- [arXiv Link](https://arxiv.org/abs/2512.01178v1)
- [PDF Link](https://arxiv.org/pdf/2512.01178v1.pdf)

## Subfields
 自动标注 (Auto-labeling) / 弱监督 3D 目标检测
## Reason for Interest

论文针对自动驾驶数据标注成本高昂的问题，提出了一种基于神经渲染（SDF）的弱监督 3D 目标检测与自动标注框架 VSRD++。其核心创新在于：1. 利用几何先验（Cuboid + Residual SDF）从单目视频序列中重建 3D 包围盒；2. 引入速度属性和时变 SDF 有效解决了动态物体的标注难题，克服了传统 NeRF/SDF 方法难以处理动态场景的缺陷。实验结果表明，该方法在仅使用 2D 分割掩码作为监督的情况下，其生成的伪标签质量甚至优于部分依赖 LiDAR 的弱监督方法，具有极高的行业应用价值，能大幅降低数据生产成本。
