# [3.8] 3D Point Cloud Object Detection on Edge Devices for Split Computing

- Authors: Taisuke Noguchi, Takuya Azumi
- [arXiv Link](https://arxiv.org/abs/2511.02293v1)
- [PDF Link](https://arxiv.org/pdf/2511.02293v1.pdf)

## Subfields
 边缘计算 / 路侧感知 (Edge Computing / Infrastructure Perception)
## Reason for Interest

1. 领域相关性：论文明确针对路侧基础设施传感器系统（Infrastructure Sensors），而非直接的车端（Vehicle-side）感知算法，属于V2X辅助范畴，根据严格评分规则，分数上限为5分。
2. 创新性：主要贡献是将通用的分割计算（Split Computing）策略应用于现有的Voxel R-CNN模型，属于系统部署层面的工程优化，缺乏核心感知算法或模型架构的实质性创新。
3. 实验验证：仅在较旧且规模较小的KITTI数据集上进行了验证，且对比基准主要是本地推理与云端推理的延迟差异，未在nuScenes或Waymo等主流现代数据集上评估，参考价值有限。
4. 实用性挑战：虽然降低了边缘端算力负载，但引入了对网络带宽和稳定性的强依赖，这在安全关键的自动驾驶感知中是显著的风险点。
