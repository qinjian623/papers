# [8.8] RCTDistill: Cross-Modal Knowledge Distillation Framework for Radar-Camera 3D Object Detection with Temporal Fusion

- Authors: Geonho Bang, Minjae Seong, Jisong Kim, Geunju Baek, Daye Oh, Junhyung Kim, Junho Koh, Jun...
- [arXiv Link](https://arxiv.org/abs/2509.17712v1)
- [PDF Link](https://arxiv.org/pdf/2509.17712v1.pdf)

## Subfields
 3D Perception / Radar-Camera Fusion
## Reason for Interest

该论文提出了一种针对雷达-相机融合的跨模态知识蒸馏框架 (RCTDistill)，创新性地设计了三个模块 (RAKD, TKD, RDKD) 分别解决传感器不确定性、动态物体的时间未对齐和特征判别性问题。论文在主流数据集 nuScenes 和 VoD 上均验证了其有效性，不仅在精度上达成 SOTA，更重要的是在保持高性能的同时显著提升了推理速度 (优于 SpaRC 等竞品)，具备极高的量产落地价值和行业潜力。
