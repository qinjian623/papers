# [8.8] OWL: Unsupervised 3D Object Detection by Occupancy Guided Warm-up and Large Model Priors Reasoning

- Authors: Xusheng Guo, Wanfa Zhang, Shijia Zhao, Qiming Xia, Xiaolong Xie, Mingming Wang, Hai Wu, C...
- [arXiv Link](https://arxiv.org/abs/2512.05698v1)
- [PDF Link](https://arxiv.org/pdf/2512.05698v1.pdf)

## Subfields
 3D感知 / 无监督目标检测 (Unsupervised 3D Object Detection)
## Reason for Interest

该论文针对自动驾驶中标注成本高昂的痛点，提出了一套高效的无监督3D检测框架。核心创新在于结合了底层的几何约束（Occupancy Guided Warm-up, OGW）与高层的语义推理（Instance-Cued Reasoning, ICR），利用大模型（LLM）的先验知识来清洗和优化伪标签。实验结果显示其性能大幅超越现有的无监督方法（如CPD, OYSTER），特别是在长尾类别（如骑行者）上提升显著（从~5%提升至~32%），证明了引入大模型推理进行伪标签过滤的有效性。虽然距离全监督方法仍有差距，但作为自动标注和数据挖掘工具具有极高的行业落地价值。
