# [4.5] LLM-Empowered Event-Chain Driven Code Generation for ADAS in SDV systems

- Authors: Nenad Petrovic, Norbert Kroth, Axel Torschmied, Yinglei Song, Fengjunjie Pan, Vahid Zolfa...
- [arXiv Link](https://arxiv.org/abs/2511.21877v1)
- [PDF Link](https://arxiv.org/pdf/2511.21877v1.pdf)

## Subfields
 大模型 / 自动驾驶代码生成 (LLM for AD Software Engineering)
## Reason for Interest

论文提出了结合形式化事件链（Event-Chain）和RAG的大模型代码生成框架，旨在解决车端软件开发中的信号幻觉和时序约束问题，具有一定的工程架构参考价值。然而，论文的实验评估非常薄弱，仅展示了一个极简单的“行人检测触发双闪”场景，缺乏对复杂自动驾驶逻辑的生成能力验证，也没有任何量化的通过率或正确性指标，无法证明其相对于现有代码生成方法的优势或在实际量产开发中的可行性。
