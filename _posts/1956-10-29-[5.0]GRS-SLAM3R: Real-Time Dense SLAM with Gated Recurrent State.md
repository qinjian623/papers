# [5.0] GRS-SLAM3R: Real-Time Dense SLAM with Gated Recurrent State

- Authors: Guole Shen, Tianchen Deng, Yanbo Wang, Yongtao Chen, Yilin Shen, Jiuming Liu, Jingchuan W...
- [arXiv Link](https://arxiv.org/abs/2509.23737v1)
- [PDF Link](https://arxiv.org/pdf/2509.23737v1.pdf)

## Subfields
 视觉SLAM / 稠密三维重建 (Visual SLAM / Dense 3D Reconstruction)
## Reason for Interest

该论文提出了一种基于门控循环状态（Gated Recurrent State）和分层子图对齐的稠密SLAM框架，有效解决了DUSt3R类方法在长序列中的漂移问题，创新性较高（将GRU机制引入Transformer潜在状态更新）。实验表明其在室内场景重建和定位精度上超越了现有的CUT3R、SLAM3R等SOTA方法。然而，该研究完全基于室内小尺度数据集（7-Scenes, Apartment等）进行验证，缺乏在自动驾驶典型户外大规模场景（如KITTI, nuScenes）下的测试。尽管稠密SLAM技术是自动驾驶感知的通用技术，但缺乏车端场景的直接适配与验证，依据评分规则（非直接车端相关最高5分），限制了其最终得分。
