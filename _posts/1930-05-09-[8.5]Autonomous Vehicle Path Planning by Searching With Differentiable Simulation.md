# [8.5] Autonomous Vehicle Path Planning by Searching With Differentiable Simulation

- Authors: Asen Nachkov, Jan-Nico Zaech, Danda Pani Paudel, Xi Wang, Luc Van Gool
- [arXiv Link](https://arxiv.org/abs/2511.11043v2)
- [PDF Link](https://arxiv.org/pdf/2511.11043v2.pdf)

## Subfields
 轨迹规划 / 仿真及端到端控制 (Trajectory Planning / Simulation-based Control)
## Reason for Interest

1. 创新性：提出DSS框架，利用可微分模拟器（Waymax）在测试时进行梯度引导的轨迹搜索与优化，并设计了分类器引导（Classifier-Guided）策略处理碰撞等不可导约束，思路新颖且逻辑自洽。
2. 效果显著：在规划任务中，ADE指标大幅优于Wayformer等主流基线，证明了利用梯度反向传播修正轨迹的有效性。
3. 局限性：虽然拟合人类轨迹（ADE）表现极佳，但在绝对安全性（碰撞率）上未完全超越所有RL基线（如PiDT），且依赖高保真可微分模拟器，Sim-to-Real的迁移难度是潜在挑战。
4. 价值：为自动驾驶中的Test-time Compute和基于模型的规划提供了高质量的实证研究，行业参考价值高。
