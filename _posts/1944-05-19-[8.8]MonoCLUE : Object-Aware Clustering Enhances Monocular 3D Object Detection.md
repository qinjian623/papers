# [8.8] MonoCLUE : Object-Aware Clustering Enhances Monocular 3D Object Detection

- Authors: Sunghun Yang, Minhyeok Lee, Jungho Lee, Sangyoun Lee
- [arXiv Link](https://arxiv.org/abs/2511.07862v1)
- [PDF Link](https://arxiv.org/pdf/2511.07862v1.pdf)

## Subfields
 单目 3D 目标检测 (Monocular 3D Object Detection)
## Reason for Interest

1. 创新性强：论文提出了一种新颖的'对象感知聚类'（Object-Aware Clustering）方法，利用 K-means 和 SAM（Segment Anything Model）引导的掩码来显式提取物体的局部视觉特征（如车顶、车头），并结合广义场景记忆（Generalized Scene Memory）来解决单目检测中常见的遮挡和截断难题。这区别于传统过度依赖深度估计几何先验的方法。
2. 实验完整且结果坚实：在主流的 KITTI 和 Waymo 数据集上均取得了 SOTA 成绩，且消融实验清晰地证明了各个模块（局部聚类、场景记忆、重定位）的有效性。
3. 行业价值：该方法直接针对自动驾驶中单目感知的长尾问题（遮挡、截断），虽然推理时间（52ms）较基线（35ms）略有增加，但在可接受范围内，且显著提升了难例（Hard case）的检测精度，具有较高的落地潜力和参考价值。
