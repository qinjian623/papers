# [8.6] FreqPDE: Rethinking Positional Depth Embedding for Multi-View 3D Object Detection Transformers

- Authors: Haisheng Su, Junjie Zhang, Feixiang Song, Sanping Zhou, Wei Wu, Nanning Zheng, Junchi Yan...
- [arXiv Link](https://arxiv.org/abs/2510.15385v1)
- [PDF Link](https://arxiv.org/pdf/2510.15385v1.pdf)

## Subfields
 Camera-based 3D Object Detection / Multi-view Perception
## Reason for Interest

该论文提出了 FreqPDE，一种针对多视角 3D 检测的频率感知位置深度嵌入方法，具有较高的研究价值和行业应用潜力。

主要优点：
1. **创新性强**：将频率域分析（小波变换）引入视觉特征金字塔（FSPE），有效结合了高频边缘信息和低频语义信息，解决了小目标和边界模糊问题。同时引入基础大模型（DepthAnything）进行混合深度监督，符合当前利用大数据增强感知的技术趋势。
2. **实验扎实**：在 nuScenes 数据集上进行了广泛实验，验证集和测试集均取得 SOTA 成绩（Test NDS 64.2%）。消融实验详细，涵盖了各模块有效性、计算开销及监督方式。
3. **工程价值**：虽然引入了额外的计算模块，但论文提供了轻量化版本（FreqPDE-L1），在保持性能提升的同时控制了推理延迟，具备实际部署潜力。

不足之处：
相比 Baseline (StreamPETR)，标准版模型的推理速度有所下降（FPS 从 10.7 降至 8.8），增加了显存占用和计算量，对端侧部署提出了更高要求。
